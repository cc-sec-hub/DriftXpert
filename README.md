**DriftXpert** is an AI-driven Network Intrusion Detection System (NIDS) framework designed to address model robustness, and adaptive defense scenarios. 

---

## 1. Prerequisites & Environment Setup

This framework requires Python 3.8.12 and PyTorch with CUDA 11.7 support. We highly recommend using a Python virtual environment (conda or venv) to manage dependencies.

### Installation

Clone the repository and install the dependencies from the `requirements.txt`:

```bash
# Clone the repository
git clone <your-repository-url>
cd driftxpert

# Install the dependencies
pip install -r requirements.txt
